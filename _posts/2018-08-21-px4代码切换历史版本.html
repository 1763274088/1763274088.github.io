PX4代码切换历史版本

克隆Firmware、新建分支并上传

克隆Firmware：

mkdir Firmware_all

cd Firmware_all

git clone https://github.com/1763274088/Firmware.git

cd Firmware

新建Firmware分支：

git checkout v1.6.5 %%处于v1.6.5head的状态，然后新建分支

git checkout -b branch_1.6.5 %%新建分支branch_1.6.5，并切换到该分支：

git push --set-upstream origin branch_1.6.5 %%上传新的分支

git submodule update --init --recursive %%更新子模块
